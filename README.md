# Quick Start

## Install Dependencies

```
npm install -g @vue/cli
npm install -g express-generator
```

## Vue.js (Frontend)

```
cd frontend
npm install
npm install vue-router --save
npm install axios --save
npm run serve
```

## Express (Backend)

```
cd backend
npm install
npm start
```

## Build Frontend

```
cd frontend
npm run build
```
